DDAU-Net: Defect Detection in Steel with U-Net

An article about the project using U-Net architecture for defect detection in steel.

Introduction

The DDAU-Net project aims to enhance the defect detection process in steel sheets using the U-Net architecture. This article provides an overview of the project, its significance, and the expected benefits.

Motivation

Steel is a fundamental material in modern construction. However, to ensure efficiency in steel production, identifying and correcting defects is crucial. DDAU-Net emerges as a machine learning solution to improve the detection of defects in steel sheets, contributing to production efficiency.

Implementation

DDAU-Net utilizes the U-Net architecture, an effective framework for image segmentation. The implementation is explored, covering data preprocessing to model training, highlighting crucial steps for accurate defect detection.
