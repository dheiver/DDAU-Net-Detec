Severstal Steel Defect Detection

This GitHub Pages website provides information about the Severstal Steel Defect Detection project.

Overview

This project involves the detection of defects in steel using the U-Net architecture in Keras. The code is designed to be extensible for experimenting with different models.

Sections

  • Preprocessing: Expand the train dataframe and create utility functions.
  • Data Generator: Generate data for Keras using a custom data generator.
  • Model Architecture: U-Net architecture for defect detection.
  • Training: Train the model with provided data.
  • Evaluation & Submission: Evaluate the model and create a submission.
